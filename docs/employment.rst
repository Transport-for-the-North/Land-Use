Employment Model
################

Model Specification Summary
===========================

The employment model provides an estimate of the number of jobs at LSOA (2021)
zone level for a given year.

As currently specified, the employment model requires the following dimensions:

- 2020 Standard Occupational Classification (SOC 2020)
- Standard Industrial Classifications (SIC 2007)
